# homebrew-schoolcode

Homebrew tap for [SchoolCode](https://github.com/luka-loehr/SchoolCode) - Automated developer tool deployment for macOS Guest accounts.

## Installation

```bash
brew tap luka-loehr/schoolcode
brew install schoolcode
```

## What it does

Installing SchoolCode via Homebrew will automatically:
- Run compatibility checks
- Perform system repairs
- Install development tools (Python, Homebrew, Git, pip)
- Configure Guest account environment

Uninstalling will automatically clean up all SchoolCode components.
